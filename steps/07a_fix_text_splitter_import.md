### **Krok 7a: Poprawka - Instalacja pakietu `langchain-text-splitters`**\n\nBłąd importu `langchain.text_splitter` wynika z tego, że komponenty do dzielenia tekstu zostały wydzielone do osobnego pakietu, aby zmniejszyć rozmiar głównej biblioteki `langchain`. Musimy go jawnie zainstalować.\n\n**Co masz zrobić:**\n\n1.  **Zaktualizuj plik `requirements.txt`:**\n    *   Otwórz plik `requirements.txt`.\n    *   Dodaj `langchain-text-splitters` do listy zależności. Plik powinien teraz wyglądać tak:\n\n    ```txt\n    django\n    langchain\n    langchain-community\n    langchain-text-splitters # <-- Dodaj tę linię\n    langgraph\n    python-dotenv\n    unstructured[docx,pdf,md,json]\n    python-magic-bin\n    chromadb\n    sentence-transformers\n    ```\n\n2.  **Zainstaluj zaktualizowane zależności:**\n    *   W terminalu z aktywnym środowiskiem wirtualnym, uruchom ponownie komendę instalacji:\n        ```bash\n        pip install -r requirements.txt\n        ```\n\n3.  **Potwierdź poprawny import:**\n    *   Po instalacji pakietu, import w pliku `ingestion/agents.py` powinien już działać poprawnie. Upewnij się, że linia importu wygląda tak:\n        ```python\n        from langchain_text_splitters import RecursiveCharacterTextSplitter\n        ```\n        Zwróć uwagę, że import teraz pochodzi z `langchain_text_splitters`, a nie `langchain.text_splitter`.
